# WiFi QR Code Generator

The **WiFi QR Code Generator** is a web application that makes sharing WiFi credentials easy and secure. Quickly generate a QR code with your WiFi network details, enabling guests to connect by simply scanning the code—no more manual password entry required.

Built with **HTML**, **CSS**, and **JavaScript**, this tool provides a responsive and user-friendly experience on both desktop and mobile devices. The clean interface allows users to input their WiFi details and create a QR code within seconds. Plus, you can download the QR code as a PNG for effortless sharing or printing.

### Features
- **SSID and Password Input**: Securely input WiFi network name (SSID) and password.
- **QR Code Generation**: Generates a QR code representing the WiFi credentials.
- **Download Option**: Download the QR code as a PNG for easy sharing or printing.
- **Responsive Design**: Works seamlessly on all devices for an optimal user experience.

### Technologies Used
- **HTML**: Provides the structure and semantic layout.
- **CSS**: Custom styles for a clean, navigable design.
- **JavaScript**: Manages form submissions, QR code generation, and dynamic content.
- **jQuery**: Simplifies DOM manipulation and enhances interaction.
- **QRCode.js**: Generates QR codes from user inputs.

### How to Use
1. **Open the Application**: Launch the WiFi QR Code Generator in a web browser.
2. **Enter WiFi Credentials**: Input your SSID and Password.
3. **Generate QR Code**: Click "Generate QR Code" to create the QR code.
4. **Download**: Click "Download QR Code" to save the image as a PNG.

### Running Locally
1. Download the source code as a zip.
2. Extract the zip file.
3. Open the `index.html` file in your browser.


### Conclusion
The **WiFi QR Code Generator** provides a modern and efficient solution for sharing WiFi access. With this application, you can quickly share your WiFi network with friends, family, or clients—making the process simpler and more tech-savvy than ever!
