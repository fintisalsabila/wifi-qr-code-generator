# WiFi QR Code Generator
Welcome to the **WiFi QR Code Generator**—a web app that makes sharing WiFi credentials easy and secure. Generate QR codes with your network information to let guests connect without manually entering passwords. This tool is perfect for sharing access with friends, family, or clients through a simple, scannable QR code containing your SSID and password.

## Features
- **SSID & Password Input**: Securely enter your WiFi network name and password.
- **QR Code Generation**: Instantly create a QR code representing your WiFi credentials.
- **Download Option**: Save the generated QR code as a PNG image for easy sharing or printing.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Technologies Used
- **HTML**: Provides the application's structure.
- **CSS**: Adds custom styling for a clean, user-friendly interface.
- **JavaScript**: Handles form submissions, QR code generation, and dynamic updates.
- **jQuery**: Simplifies DOM manipulation and event handling.
- **QRCode.js**: Library used to generate QR codes from user inputs.

## Getting Started
1. Download and unzip the source code.
2. Open the HTML file in your browser, and you're ready to use the app!

## Usage
1. **Open the Application**: Launch the app in your browser.
2. **Enter WiFi Details**: Input your SSID (WiFi name) and password.
3. **Generate QR Code**: Click "Generate QR Code" to create the QR.
4. **Download**: Save the QR code by clicking "Download QR Code."

## Conclusion
The WiFi QR Code Generator is a quick, modern solution for sharing network access without hassle. Powered by HTML, CSS, and JavaScript, it provides a seamless experience for generating and sharing QR codes—making it the smart choice for simplified WiFi sharing.
